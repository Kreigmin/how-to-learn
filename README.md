# Научиться учиться
## Описание проекта
Это одностраничный сайт о проблемах получения знаний и примерах решения этих проблем, созданный на чистом HTML и CSS. Сайт состоит из шапки, девяти секций с контентом и подвала.  Реализованы:
* простая анимация через @keyframes;
* flex;
* прозрачность при наведении на ссылку;
* видео с Youtube.
## Использованные технологии
* HTML5;
* CSS3.
## Что можно улучшить
В секцииях **digits** и **kaufman** таблицу можно сделать через ``` <ul> ``` и ``` <li> ``` вместо ``` <div> ```, чтобы соблюсти семантику.
